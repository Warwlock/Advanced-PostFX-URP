## \[0.1.2] - 2025-11-16

### Added
- Effects table is added in documentation.
- Added "enabled" property to the GTAO and ExtendedDOG effects.


### Changed
- Default values of GTAO, AutoExposure and ToneMapping effects are changed for ease of use.
- Name of "isEnabled" property changed to "enabled" for AutoExposure effect.
